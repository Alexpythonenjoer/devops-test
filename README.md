# DevOps тестовое задание

## Описание проекта

Данный проект представляет собой простое веб-приложение, развёрнутое с помощью Docker и Docker Compose.  
Оно состоит из двух сервисов:

- **backend** – HTTP-сервер на Python, который возвращает строку `"Hello from Effective Mobile!"` по пути `/`.
- **nginx** – обратный прокси-сервер, который принимает запросы от пользователя на порту 80 и перенаправляет их на backend-сервис.

Взаимодействие между сервисами организовано через изолированную Docker-сеть. Backend не доступен напрямую с хоста, все запросы проходят через nginx.

---

## Архитектура
````
Пользователь
│
▼
localhost:80 (nginx)
│
│ proxy_pass http://backend:8080
▼
backend (Python HTTP-сервер, порт 8080)
│
▼
Ответ: "Hello from Effective Mobile!"
````


**Технологии:**
- Python 3.11
- Nginx
- Docker & Docker Compose

## Запуск проекта

### Требования
- Установленный Docker
- Установленный Docker Compose 
- Свободный порт 80 на хост-машине

### Инструкция

1. **Клонируйте репозиторий:**
   ```bash
   git clone <url-репозитория>
   cd devops-test
2. **Запустите контейнеры в фоновом режиме:**
   ````bash
   docker-compose up -d
   ````
### Проверка работоспособности
**Выполните команду:**
   ````bash
   curl http://localhost
   ````
**Ожидаемый ответ:**
   ````
   Hello from Effective Mobile!
   ````
